# Now lets use the hook

We have now exposed the size object.

<CodeSnippet code={`const [size, sizeRef] = useSize();`} />

And for this example we will render the size of the Box
in a `<pre>` element below.

<CodeSnippet
  code={`
<pre>
  {JSON.stringify(size, null, 2)}
</pre>
`}
/>

Make sure to resize the view port to see the numbers change

Note the difference between height and borderBoxSize.blockSize
and between width and borderBoxSize.inlineSize.
