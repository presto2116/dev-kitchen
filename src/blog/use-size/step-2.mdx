# Adding the hook

Now its time to add the hook. We must first import it.

<CodeSnippet code={`import useSize from '@hzdg/use-size';`} />

Then initialize it.

<CodeSnippet code={`const [, sizeRef] = useSize();`} />

and add the a ref to the element you want to know the size of

<CodeSnippet code={`<Box ref={sizeRef}>`} />

(Note: visually you will not see any changes in the rendered result)
