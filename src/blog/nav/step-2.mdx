# Now add the hook

## Import

First we import the use size hook

<CodeSnippet code={`import useSize from '@hzdg/use-size';`} />

## Initialize

then we initialize the hook to expose the width and ref

<CodeSnippet code={`const [{width}, ref] = useSize();`} />

(Note: width is only one attribute exposed by useSize())

<Link to={'/blog/use-size/'}>Click here for more info</Link>

<br />
<br />

## Attach ref

lastly we attach the ref to the element we want to get the size of

<CodeSnippet
  code={`
<Flex
  ref={ref}
  sx={{
  flex: '0 1 800px',
  justifyContent: 'space-between',
  border: (theme: Theme) => \`3px solid \${theme?.colors?.green}\`,
  }}
>`}
/>

(Note we wrapped the entire component in a Box so we can output our state
in the result)
Below the nav list in the Result tab, you will now see the text "width: [current width in pixels]"
