# Hamburger time

Only thing missing on this nav is a tasty hamburger.

Add a state hook to trigger dropdown active

<CodeSnippet
  code={`const [dropdownActive, setDropdownActive] = useState<boolean>(false);`}
/>

Add visibility attribute and toggle when dropdownActive

<CodeSnippet
  code={`
const dropdownStyles = {
  flexDirection: 'column',
  visibility: dropdownActive ? 'visible' : 'hidden',
  position: 'absolute',
  top: '100%',
};`}
/>

Add lastly, add that Menu Button

<CodeSnippet
  code={`
{isDropdown && (
  <MenuButton onClick={() => setDropdownActive(!dropdownActive)} />
)}`}
/>

And there we have it, a functionable nav that can adjust with data provided.
