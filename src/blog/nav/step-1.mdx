## Flex Element Styles

<CodeSnippet
  code={`
<Flex
  sx={{
    flex: '1 1 auto',
    justifyContent: 'space-between',
    border: (theme: Theme) => \`3px solid \${theme?.colors?.green}\`,
  }}
>`}
/>

`flex: 1 1 auto;`
stretches the flex div to fill space

`justifyContent: space-between;`
spaces the items evenly and border adds a visual border so you can see what we are doing

## Link Styles

`variant={'nav'}` has some base styling hidden inside the theme some useful attributes to know for this post are

<CodeSnippet
  code={`
nav: {
  flex: '0 0 auto',
  whiteSpace: 'nowrap',
  fontSize: 3,
  p: 2,
  textAlign: 'center',
  textDecoration: 'none',
  bg: 'grey',
  color: 'white',
  transition: 'background 0.3s ease-in-out',
  '&.active, &.active:hover': {bg: 'boulder'},
  '&:hover': {bg: 'boulder'},
},`}
/>

Most important is whiteSpace: 'nowrap' this means our link labels, nomatter the length, will never wrap to a second line
