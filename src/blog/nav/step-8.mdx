# Adding dropdown styles to our list

Let's add some conditional styling. We renamed
renderedStyles to rowStyles and added new dropdownStyles

<CodeSnippet
  code={`
const rowStyles = {
  flex: '0 1 800px',
  justifyContent: 'space-between',
  border: (theme: Theme) => \`3px solid \${theme?.colors?.green}\`,
};
const dropdownStyles = {
  flexDirection: 'column',
  border: (theme: Theme) => \`3px solid \${theme?.colors?.green}\`,
};`}
/>

Now you can see when isDropdown is true, our list will use dropdown styles

<CodeSnippet
  code={`const renderedStyles = isDropdown ? dropdownStyles : rowStyles;`}
/>
